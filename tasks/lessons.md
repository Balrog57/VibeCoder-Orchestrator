# Leçons Apprises (Memory)

## Concepts et Patterns
- L'utilisation de `execa` avec arguments en tableau est cruciale pour éviter les injections de commandes.
- Utiliser un fallback vers `codex-cli` si `gemini-cli` échoue.
- Format strict de réponse Tech Lead via expressions régulières "### FILE:" et "### RUN:".
